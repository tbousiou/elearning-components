üé® Customization Options

Aspect Ratio Control

Modify the padding-top value in .flipcard-content:

.flipcard-content { padding-top: 125%; /* Current: Tall cards (4:5 ratio) */ /* padding-top: 100%; Square cards (1:1 ratio) */ /* padding-top: 75%; Wide cards (4:3 ratio) */ }

Color Themes

Add theme classes for different color schemes:

.flipcard.theme-blue .flipcard-front { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); } .flipcard.theme-green .flipcard-front { background: linear-gradient(135deg, #48bb78 0%, #38a169 100%); }

Grid Spacing

.flipcard-grid { gap: 1.5rem; /* Adjust spacing between cards */ padding: 2rem 1rem; /* Adjust container padding */ }
